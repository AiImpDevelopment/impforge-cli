# Energy Web (Vue 3.5)

Vue 3.5 + Tailwind 4 + ark-ui scaffold for utility customer portals with billing, outage maps, usage dashboards, and Green Button data export.

**Industry:** `energy`  ·  **Category:** `web`  ·  **Framework:** `vue-3.5`  ·  **License:** `MIT`

## Run locally

```bash
bun install && bun run dev
```

The preview is reachable at `http://localhost:5173` once it's ready.

## Production build

```bash
bun run build
```
