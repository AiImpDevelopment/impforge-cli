# Example — Todo App

A reference template demonstrating the [`template.json` v1 spec](https://github.com/AiImpDevelopment/impforge-mcp-manifests/blob/main/spec/template.json.v1.md). Use this as a starting point when you build your own templates.

## Run it standalone (no ImpForge required)

```bash
git clone https://github.com/AiImpDevelopment/impforge-templates
cd impforge-templates/templates/example-todo-app
bun install
bun run dev
# open http://localhost:5173
```

## Run it via ImpForge (paid Pro tier)

```bash
# in HyperChat
"scaffold me an example todo app"
```

ImpForge spawns a pop-out preview window automatically, customises via natural language, and audits the result.
