# Nonprofit Web (Vue 3.5)

Vue 3.5 + Tailwind 4 + ark-ui scaffold for nonprofit donor portals, campaign landing pages, and impact reports with IRS Schedule O transparency.

**Industry:** `nonprofit`  ·  **Category:** `web`  ·  **Framework:** `vue-3.5`  ·  **License:** `MIT`

## Run locally

```bash
bun install && bun run dev
```

The preview is reachable at `http://localhost:5173` once it's ready.

## Production build

```bash
bun run build
```
