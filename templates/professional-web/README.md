# Professional Services Web (Astro 5)

Astro 5 + Tailwind 4 scaffold for consultancies, agencies, and accounting firms with content-driven marketing pages and embedded interactive booking widgets.

**Industry:** `professional`  ·  **Category:** `web`  ·  **Framework:** `astro-5`  ·  **License:** `MIT`

## Compliance

> GDPR, SOC2, ISO-27001, EPRIVACY, CCPA

The full machine-readable rule set lives next to this README at `compliance-rules.json` and conforms to the [`compliance-rules.json` v1 spec](https://github.com/AiImpDevelopment/impforge-mcp-manifests/blob/main/spec/compliance-rules.json.v1.md).

## Run locally

```bash
bun install && bun run dev
```

The preview is reachable at `http://localhost:5173` once it's ready.

## Production build

```bash
bun run build
```
